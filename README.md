# genesis
a 65% keyboard that's a year in the making   
> 50% learning cad/keyboard design, 30% doing nothing, 20% actually making the keyboard
## features
* 7 deg angle
* top mount
* seamless
* arrow blocker
* weight
* support for a bakeneko65 pcb (untested but should work in theory)
